About My Projects

Git 'er Done is my first collaborative project. I worked with three other developers to create this browser-based app that rewards productivity with funny gifs.

The user creates daily to-dos. As each one is checked off, a light-hearted gif will appear as a congratulatory gesture. This will be an on-going project for me. I plan to add a timer to the to-dos; if the task isn't completed in a certain amount of time, a not-so-funny gif could appear to the user. This project could become a mobile app, as well.

Besides Git 'er Done, the other placeholders on this page are for links to my future projects I look forward to completing. Please check out my Github repository page for other full-stack development assignments I've completed. Thank you for visiting!
